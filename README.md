Why Data Cleaning is Crucial in This CRM Analysis
Data cleaning is a critical first step in CRM analysis because decisions made from dirty data are unreliable and can lead to wrong marketing strategies.
In this project, data cleaning was necessary because:
Missing values (e.g., income, spending, loyalty score) can distort averages and cluster formation.
Inconsistent formats (dates, categorical values, text fields) affect calculations such as recency and campaign response.
Duplicate customers inflate customer counts and overestimate performance.
Outliers (e.g., extremely high spending values) can bias cluster centroids in K-means and misclassify customers.
